###### 准备工作
###### 1 安装nodejs 
###### 2 npm install

###### 开发
###### npm run start     开启文件监听&本地起服务

###### 发布
###### npm run start
###### npm run build     生成压缩文件

##  目录

###### src/  源码目录             所有修改都在此目录下
###### dest/ 编译源码目录          此目录在开发模式下自动生成，浏览器查看此目录
###### dist/ 线上发布目录          压缩混淆代码
###### WEB-INF/server.conf       开发环境代理配置文件
